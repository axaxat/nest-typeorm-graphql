
## Description

Знакомство с graphql. 

### Особенности:

Корневой узел может быть только один.

Удаление узла каскадное.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Запуск с базой(нужен docker) http://localhost:3000/graphql 
$ npm run start:withDB
# Останавливает работу DB
$ npm run stop:withDB
```
