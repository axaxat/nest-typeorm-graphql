
## Description

Знакомство с graphql.

### Особенности:

Корневой узел может быть только один.

Удаление узла каскадное.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Запуск с базой(нужен docker) http://localhost:3000/graphql 
$ npm run start:db
$ npm run start
# Останавливает работу docker-compose
$ npm run stop:db
```
